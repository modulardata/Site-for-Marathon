Description

In the previous stage, you added YouTube and weather iframes to the landing page. Now, change the style of these iframes using CSS so that they could fit the page perfectly.

Position the iframes so that they follow each other and take up 100% of the screen width without margins or padding.

Using relative units vw with the iframe-container should help!

Also, adjust the height of the YouTube iframe to be 80% of the available screen height while keeping the width of both frames. This will help users understand that our branding is not limited to just the video. You can use the overflow and position properties in your CSS code to position the elements.

Finally, let's remove unnecessary controllers from the YouTube iframe. controls=0 will help you.

Objectives

  • Set the width of both iframes to 100% of the screen width;

  • Set the height of the YouTube iframe to 80% of the screen height;

  • Remove any margin and padding from the body and iframes;

  • Hide the controls for YouTube iframe.
